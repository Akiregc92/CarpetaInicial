Formas de Usar CSS

  1. Enlace dentro del HEAD a un archivo .css , con la etiqueta link rel="stylesheet" href="style.css"
  2. Etiqueta Style dentro del HEAD, ahi se implementa el css, no es una buena forma porque demora en cargar la pagina
  3. Estilo embebido ,Como atributo Style para una etiqueta html , por ejemplo para la etiqueta p style = "color:red;"
